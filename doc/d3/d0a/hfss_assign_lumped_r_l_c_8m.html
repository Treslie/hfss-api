Creates the necessary VB Script to assign a Lumped LRC to a given Object.

Note
Every parameter is mandatory. If any of the three LRC conditions is set to false, its correspondent value and unit are not read.
Example
1 fid = fopen('myantenna.vbs', 'wt');
2 ...
3 hfssAssignLumpedRLC(fid, 'RLC1', 'Sheet', true, true, false, 10, 2, 0,...
4  [-width/2, 0, 0], [width/2, 0, 0], {'ohm', 'nH, ...
5  'pF', 'mm'});
Author

Date

Parameters
fidfile identifier of the HFSS script file.
Namename of the lumped RLC (appears under Boundaries in HFSS).
ObjNamename of the (sheet-like) object to which the floquet port is to be assigned.
UseResist(boolean) enables Lumped RLC boundary as Resistance
UseInduct(boolean) enables Lumped RLC boundary as Inductance
UseCap(boolean) enables Lumped RLC boundary as Capacitance
ResistResistance value
InductInductance value
CapCapacitance value
iStart(vector) starting point of the Current Flow Line. Specify as [x, y, z].
iEnd(vector) ending point of the Current Flow Line. Specify as [x, y, z].
Units(cell vector){resistance, inductance, capacitance, distance} specifies units for Resistance, Capacitance, Inductance and position vectors.
  • Possible values for Resistance units are uOhm, mOhm, ohm, kOhm, megohm and GOhm.
  • Possible values for Inductance units are fH, pH, nH, mH and H.
  • Possible values for Capacitance units are fF, pF, nF, mF and farad.
