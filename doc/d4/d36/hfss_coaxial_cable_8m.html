Creates a Coaxial Cable.

This function creates the VBScript necessary to draw a Coaxial Cable in HFSS. The "Coaxial Cable" can have as many cylinders as you specify. This function only creates the geometric structure and does not set any material properties.

Example
1 ...
2 hfssCoaxialCable(fid, {'Cyl1_In', 'Cyl1_Er', 'Cyl1_Out'}, 'Z', [0, 0, 0],
3  0.02, 0.03, 0.04], 10, 'in')
Parameters
fidfile identifier of the HFSS VBScript File.
Namesa cell of strings that contains the names of each cylinder that is a part of the co-axial cable (see example).
Axisaxis of the Coaxial Cable (choose between X, Y or Z).
CenterCoordinates of the Center of the Coaxial Cable ([x, y, z]).
Radiian array of Radii of the Cylinders present in the Coaxial cable. Each radius corresponds to the respective name specfied in the Name parameter.
Heightlength of the coaxial cable.
Unitscan be either in or mm or meter or anything defined in HFSS.
