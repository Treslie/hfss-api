Creates a new report with a single trace and adds it to the Results branch in the project tree.

Note
This function has to be used AFTER a solution is inserted with hfssInsertSolution.
Example
1 fid = fopen('myantenna.vbs', 'wt');
2 ... create some objects here ...
3 ... insert far field spheres ...
4 hfssInsertFarFieldSphereSetup(fid, 'EPlaneCutSphere', [0 180 1], [90 90 1]);
5 hfssInsertFarFieldSphereSetup(fid, 'HPlaneCutSphere', [90 90 1], [0 180 1]);
6 
7 hfssCreateReport(fid, 'E Plane Cut (cart.)', 5, 1, 'Solution', [],...
8  'EPlaceCutSphere', [], {'Theta', 'Phi', 'Freq'},...
9  {'Theta', 'dB(DirTotal)'});
10 hfssCreateReport(fid, 'E Place Cut (polar)', 5, 3, 'Solution', [],...
11  'EPlaneCutSphere', [], {'Theta', 'Phi', 'Freq'},...
12  {'Theta', 'dB(DirTotal)'});
13 hfssCreateReport(fid, 'H Plane Cut (cart.)', 5, 1, 'Solution', [],...
14  'HPlaneCutSphere', [], {'Phi', 'Theta', 'Freq'},...
15  {'Phi', 'dB(DirTotal)'});
16 hfssCreateReport(fid, '3D Diagram Cart.', 5, 6, 'Solution', [],...
17  'Diagram3D', [], {'Theta', 'Phi', 'Freq'},...
18  {'Theta', 'Phi', 'dB(DirTotal)'});
19 hfssCreateReport(fid, '3D Diagram Polar', 5, 7, 'Solution', [],...
20  'Diagram3D', [], {'Phi', 'Theta', 'Freq'},...
21  {'Phi', 'Theta', 'dB(DirTotal)'});
22 hfssCreateReport(fid, 'Return Loss', 1, 1, 'Solution1', 'Sweep1',...
23  [], 'Sweep', {'Freq'}, {'Freq',...
24  'db(S(LumpP1,LumpP1))'});

Parameters
fidfile identifier of the HFSS script file.
ReportNamename of the report.
Typetype of the report (integer 1-6). Possible values are:
  • 1: "Modal S Parameters"
  • 2: "Terminal S Parameters"
  • 3: "Eigenmode Parameters"
  • 4: "Fields"
  • 5: "Far Fields"
  • 6: "Near Fields"
  • 7: "Emission Test"
Display
  • if Type is 1-3, then Display's possible values are:
    • 1: "Rectangular Plot"
    • 2: "Polar Plot"
    • 3: "Radiation Pattern"
    • 4: "Smith Chart"
    • 5: "Data Table"
    • 6: "3D Rectangular Plot"
    • 7: "3D Polar Plot"
  • if Type is 4, then the possible values are:
    • 1: "Rectangular Plot"
    • 2: "Polar Plot"
    • 3: "Radiation Pattern"
    • 5: "Data Table"
    • 6: "3D Rectangular Plot"
  • if Type is 5-6, then the possible values are:
    • 1: "Rectangular Plot"
    • 3: "Radiation Pattern" (Polar)
    • 5: "Data Table"
    • 6: "3D Rectangular Plot"
    • 7: "3D Polar Plot"
  • if Type is 7, then the possible values are:
    • 1: "Rectangular Plot"
    • 5: "Data Table"
Solutionname of the solution given to hfssInsertSolution.
Sweepname of the frequency sweep given to hfssInterpolatingSweep. This can be an empty string, in which case it will be used LastAdaptive instead.
Contextcontext for which the expression is being evaluated. This can be an empty string if there is no context. e.g. "Infinite Sphere", "Sphere", "Polyline"
Domaindomain for which th expression is being evaluated. This can be an empty string if there is no context. e.g. "Sweep" or "Time"
VarObjcell array of names of the variables to be used as sweep definitions for the report, including the frequency/ies. The first one will be the primary sweep.
