Creates the VB Script necessary to assign a length-based mesh operation to one object.

Example
1 fid = fopen('myantenna.vbs', 'wt');
2 ...
3 hfssAssignLengthOp(fid, 'Mesh1', 'Air', Lambda/2, 'mm', true);
Author
Daniel R. Prado, danys.nosp@m.an@g.nosp@m.mail..nosp@m.com / drpra.nosp@m.do@t.nosp@m.sc.un.nosp@m.iovi.nosp@m..es
Date
04 November 2012
Parameters
fidfile identifier of the HFSS script file.
Namename of the mesh operation.
Objectobject to which the mesh operation needs to be applied.
MaxLengthmax length of the edges of triangle or tetrahedra.
Unitsunits of MaxLength, e.g. mm.
Inside(optional) Boolean which indicates if the mesh operation will be applied to the surface of the object (MaxLength restricts the edges of the triangles) or to the volume (MaxLength restricts the edges of the triangles and tetrahedra). Default is false.
RestrictElem(optional) boolean which indicates if there should be a restriction in the number of elements. If true, MaxElem should be specified. Default is false.
MaxElem(optional) Maximum number of elements to be created by the mesh if RestrictElem if true. By default is 1000.
