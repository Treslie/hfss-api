Creates the VB Script necessary for unite a given set of objects already present in the HFSS 3D Modeler.

Example
1 fid = fopen('myantenna.vbs', 'wt');
2 ...
3 // Method 1.
4 hfssUnite(fid, {'Object1', 'Object2', Object3'});
5 // ... or Method 2.
6 hfssUnite(fid, {'Object1'}, {'Object2'}, {'Object3'});
Parameters
fidfile identifier of the HFSS script file.
vararginNames of the objects that need to be united. It can be specified either as a single cell array of strings, or several individual cell strings containing a single object name (see example ...)
