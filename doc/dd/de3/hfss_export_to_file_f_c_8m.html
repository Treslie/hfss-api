Evaluates the top stack element at a set of points specified in an external file (PTS) and exports the data to a file (Name).

Note
  • This function is bad documented in scripting.pdf. It only indicates two arguments while in reallity it requires more.
  • A grid can be generated by the function hfssCreatePTS.
Example
1 hfssCreatePTS(fid, 'MyGrid', [0 0 0], [1 1 1], [.1 .1 .1], 'mm');
2 hfssEnterQty(fid, 'E');
3 hfssExportToFileTC(fid, 'EField', 'MyGrid.pts', 'Solution', Freq);
