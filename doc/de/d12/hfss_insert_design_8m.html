Create the necessary VB Script to insert an HFSS Design into the Project and set it as the active design.

Note
This function is usually called after a call to either hfssNewProject() or hfssOpenProject(), but this is not necessary.
Example
1 fid = fopen('myantenna.vbs', 'wt');
2 ...
3 hfssInsertDesign(fid, 'Dipole_SingleElement');
Parameters
fidfile identifier of the HFSS script file.
designNamename of the new design to be inserted.
designType(Optional String) choose from the following:
  1. driven modal (default)
    1. driven terminal
    2. eigenmode
